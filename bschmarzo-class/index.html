Building an Interactive Cancer Policy Simulator for Iowa using LLMs.

🚀 Launch Live Simulator

The Challenge

"I want to create a simple interactive widget for my students... Cancer in Iowa is a major issue. Please research all credible sources about the cancer situation in Iowa and in similar states to help build this widget."

The Six Strategic Levers

1. Cigarette Tax

High impact on lung cancer; generates immediate state revenue.

2. Pesticide Tax

Targets chemical runoff; affects agricultural production costs.

3. Cover Crop Subsidies

Reduces nitrate runoff in water; high upfront cost for farmers.

4. Radon Mitigation

Iowa has high natural radon levels; requires home-by-home fixes.

5. Ethanol Regulation

Addresses industrial emissions; high economic friction.

6. Rural Mobile Screenings

Increases early detection in healthcare-sparse rural areas.

The Master Prompt

Copy and paste the exact text below into your LLM to generate the simulator:

Act as a Lead Web Developer. Create a single, self-contained HTML file that includes CSS and JavaScript. This file should be an interactive 'Iowa Cancer Policy Simulator.'

The UI:
- A clean, modern dashboard with a 'Remaining Budget' counter starting at $50.00.
- Six sliders representing: 1. Cigarette Taxes, 2. Pesticide Taxes, 3. Cover Crop Subsidies, 4. Radon Mitigation, 5. Ethanol Regulation, 6. Rural Mobile Screening.
- A 'Total Economic Value' meter that updates in real-time.

The Logic:
- Each slider should have an assigned 'Cost per Unit' and an 'Economic Benefit Multiplier' (based on a 1-10 scale).
- If the user tries to spend more than $50, the sliders should lock or show a warning.
- Provide a 'Download Report' button that lets the student save their configuration as a text file.

The Output:
- Deliver the response as a single, complete block of HTML code that I can save as an .html file and run in any browser. Ensure it is mobile-responsive using a simple CSS framework like Pico.css or Tailwind via CDN.
