Building an Interactive Cancer Policy Simulator for Iowa using LLMs.

The Challenge

Iowa currently faces a critical health crisis: it has the second-highest cancer incidence rate in the U.S. and is the only state where that rate is rising.

This exercise teaches students "Vibe Coding"—the art of using natural language and intent to direct an AI to build a complex, constrained optimization tool without manually writing code.

The Strategic Levers

Students must allocate a limited $50 budget across these six conflicting options to maximize long-term economic value.

Policy Action Economic Benefit (1-10) Relative Cost
Cigarette Tax Increase8.5Low
Pesticide Use Restrictions6.0Medium
Cover Crop Subsidies7.5High
Radon Mitigation Grants9.0Medium
Ethanol Plant Regulation5.5High
Mobile Rural Screenings8.0Medium

The "Mega-Prompt"

Copy the text below into an LLM (Gemini, Claude, or ChatGPT) to regenerate this tool from scratch:

"Act as a Lead Web Developer. Create a single, self-contained HTML file for an 'Iowa Cancer Policy Simulator.'

The Setup: Six sliders (Cigarettes, Pesticides, Cover Crops, Radon, Ethanol, Screenings).
The Logic: $50 total budget. Each slider has a specific cost. If spend > $50, show a warning. Update an 'Economic Benefit Score' in real-time.
The Style: Clean, mobile-responsive, and modern UI. Include a 'Download Results' button."
